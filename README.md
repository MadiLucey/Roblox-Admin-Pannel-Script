# Roblox-Admin-Pannel-Script
This is a very basic admin panel I made for some of my games.
THIS PANEL DOES NOT INCLUDE ALL COMMANDS IT ONLY INCLUDES THE FOLLOWING.
Fly me, Unfly me, Invisible me, Visible me, Btools, Jump, Speed


ReplicatedStorage
└─ AdminRemote (RemoteEvent)

ServerScriptService
└─ AdminServer (Script)

StarterGui
└─ AdminGui (ScreenGui)
   ├─ MainFrame
   ├─ FlyButton
   ├─ InvisButton
   ├─ BToolsButton
   └─ LocalScript
